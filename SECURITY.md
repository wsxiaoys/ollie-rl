# Security Policy

## Supported Versions

`ollie-rl` is pre-1.0 and under active development. Only the `main` branch
and the most recent tagged release receive security fixes.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| latest tagged release | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you believe you've found a security issue in `ollie-rl`, please report it
privately via one of the following channels:

- **Email:** meng@tabbyml.com
- **GitHub:** [Open a private security advisory](https://github.com/wsxiaoys/ollie-rl/security/advisories/new)

Please include as much of the following information as possible:

- A description of the issue and its impact.
- Steps to reproduce, ideally a minimal proof of concept.
- The affected commit / image tag.
- Your suggested fix, if you have one.

We will acknowledge your report within **3 business days** and aim to provide
an initial assessment within **7 business days**. Once a fix is ready we will
coordinate disclosure with you.

Thank you for helping keep `ollie-rl` and its users safe.
